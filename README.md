# Kaggle Python Data Science Foundation

## Learning Goals

- Learn Python fundamentals for data science
- Practice data analysis with Pandas
- Create visualizations using Matplotlib and Seaborn
- Build beginner-friendly machine learning portfolio projects

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Kaggle Notebook

Basic Python syntax, variables, functions, conditions, loops, lists, and dictionaries.

### 2. Pandas Basics

### 3. Data Visualization

Basic charts using Matplotlib and Seaborn.

Exploratory data analysis on the Titanic dataset using Python, Pandas, and Seaborn.
